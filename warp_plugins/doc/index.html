WARP Documentation

WARP (Warband Advanced Runtime Patcher) is an experimental tool that pushes the boundaries of modding for Mount and Blade: Warband with WSE2. It allows you to modify the game engine in real-time, essentially changing how the game functions on the fly.

WARP achieves this by utilizing a lightweight compiler (TCC) to compile itself from sources and execute C-code dynamically, a process known as JIT-compilation. By analyzing debug information (a *.pdb file that comes with each version of WSE2), WARP can partially reconstruct the game's source code. This, combined with dynamic compilation, enables deep-level alterations to the engine's behavior.

The plugin system makes it easy to manage these modifications. Each "patch" is a separate plugin, allowing you to enable, disable, and configure them independently through a user-friendly GUI manager.
